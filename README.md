-Bootstrap & SASS


## Features

- Google authentication
- Responsive Design


## Run Locally

Clone the project

```bash
  git clone https://github.com/zeus97/askvault.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
